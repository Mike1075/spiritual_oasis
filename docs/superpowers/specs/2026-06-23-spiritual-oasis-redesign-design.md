## 3. 视觉系统：① 流动的意识 (Liquid Consciousness)

**全新方向，不延续旧的「禅意赛博 / 星云紫极光绿」基因。** 用 2026 当下设计语言 + Fable 5 的 WebGL 能力做"很美、很有未来科技感"。

### 3.1 气质

- **Hero = 全屏流体 / 墨流 shader**：缓慢呼吸流动的发光流体，隐喻"意识 / 源头"。最终用 **Fable 5** 生成 Three.js / 自定义 shader / 流体模拟。
- **底盘**：深邃高级的**暖黑底**（非蓝调赛博），单一会"呼吸"的光色渐变。
- **字体**：动态可变字体（kinetic type）大标题，随滚动微动；正文干净无衬线；中文标题选高质量字形。
- **三支柱**：**玻璃拟态卡**浮在流体之上；每支柱微调色温（① 冷调微光 / ② 暖调大地墨 / ③ 柔光暖白）以区分。
- **转场**：页面间用 **View Transitions API**；区块入场用**原生滚动驱动动效**（CSS scroll-timeline）。
- **细节**：mesh 渐变光晕、细颗粒(grain)。
- **可访问性**：所有动效带 `prefers-reduced-motion` 静态兜底（流体退化为静态渐变图）；性能预算受控（移动端降级）。

### 3.2 生图 (text-to-image)

- 风格统一关键词：**留白、光感、流动、东方意境、不堆元素**（信任大模型自然排版，不堆色值/参数）。
- 用 codex 生图产出：Hero 备用静态图、课程封面、关于页配图。

## 6. 技术栈

- **框架**：Next.js 16 App Router + React 19 Server Components（沿用现仓库）。
