## 1. 目标
把小鹅通全店课程**完整纳入官网"心灵大学"**，拆成独立的精美课程卡、按支柱分类呈现。本期**只做付费课**；免费课（资料在微信公众号）后做。支付 v1 上线前，付费课"购买"先外链小鹅通；上线后无缝切站内购买。

## 2. 已锁定决策（与用户确认）
1. **保留现有 3 大支柱**（面向未来教育 / 心灵成长 / 陪伴服务），把课塞进去；大支柱内部用**二级小组**撑住课量、不臃肿。
2. **商业/AI/创富课归「面向未来教育」**（"以 AI 为助力"语义）。
3. **付费课卡片"购买"近期外链小鹅通**该课详情页（已抓到每门 `jump_url`）；支付 v1 上线后切站内。
4. **卡片走信息丰富版**：封面图 + 标题 + 一句话简介 + 形式/时长徽章 + 现价 + 划线原价 + 销量/已报名人数。

## 3. 数据模型（重构 `src/data/academy.ts`）
保留 `Pillar`/`Course` 思路，扩展字段；课程数据从 `docs/xiaoe-details/xiaoe-all-goods-merged.json` 灌入。

```ts
type CourseFormat = "online" | "retreat" | "app" | "service" | "video" | "live" | "camp" | "reading";
type Course = {
  id: string;
  titleZh: string; titleEn: string;
  blurbZh: string; blurbEn: string;
  cover: string;                 // /images/academy/<id>.jpg（codex 生成）
  format: CourseFormat;
  subgroup?: string;             // 二级小组 id（仅心灵成长用，见 §4）
  // 商务字段（付费课）
  priceFen?: number;             // 现价（分）
  originalPriceFen?: number;     // 划线原价（分）
  salesCount?: number;          // 销量/已报名（小鹅通抓取值，标注"截至抓取日"）
  durationLabelZh?: string;      // 如 "30 章" / "7 天" / "3 阶"
  // 来源 / 跳转
  source: "internal" | "xiaoe" | "external";
  href: string;                  // internal=站内路由；xiaoe=小鹅通 jump_url；external=其它子站
  xiaoeProductId?: string;       // p_xxx / course_xxx（对账用）
  productKey?: string;           // ★为支付 v1 预留：上线后据此切站内购买，不迁数据
  recruiting?: boolean;
};
```
- `subgroup` 只在"心灵成长"支柱用；渲染时该支柱按 subgroup 分小节，其余支柱平铺。
- `salesCount` 注明"截至 2026-06-25 抓取"，避免显示成实时数据造成误导。

## 5. 卡片设计（升级 `CourseCard.tsx`）
信息丰富版，单卡含：
- 封面图（16:10，codex 生成；缺图回退渐变）
- 徽章行：形式/时长（线上录播/线下闭关/直播/训练营 + "30章""7天""3阶"）、`recruiting`"在招"、付费课可加"小鹅通"来源标
- 标题 + 一句话简介
- 价格行：**¥现价**（大）+ ~~¥原价~~（划线灰）+ "已售/已报名 N"（小、注抓取日）
- CTA：`source=xiaoe` →「前往购买」(新标签开小鹅通)；`internal` →「了解 / 报名」(站内)；免费课 →「免费学习」
- 视觉延续现有暗色玻璃拟态 + 青金/暖光描边

## 6. 目录页（升级 `AcademyCatalog.tsx`）
- 顶部：标题 + 支柱锚点导航 +（课量大）轻量**筛选条**（按支柱 / 形式 / 价格段：免费·≤99·100-2999·3000+）
- 主体：3 支柱分区；"心灵成长"内按 subgroup 出二级小标题网格；其余支柱平铺
- 移动端单列、桌面 3 列

## 7. 配图（codex 文生图）
- 用 `text-to-image` 技能（codex 内置 image_gen，走订阅额度）批量生成约 **26 张**付费课封面 + 若干新支柱图。
- **统一风格基准**：心灵家园美学——青金(teal/gold)、晨光暖调、东方禅意 + 微光，抽象意境为主、避免堆字（封面文字交给卡片渲染，不烧进图）。每门按主题定一句 prompt（如道德经→水墨太极山水微光；AI课→金色光网×东方意象）。
- 落盘 `/public/images/academy/<id>.jpg`，文件名对齐 Course.id。

## 8. 与支付 v1 对齐
- 每门付费课带 `productKey`（如 `xe-daodejing`）。
- 支付 v1 上线后：在 `course_sessions` 建对应场次/价格 → 卡片 CTA 从外链小鹅通切到站内 `/courses/<key>` 详情 + 购买；`academy.ts` 改 `source: "internal"` 即可，**不动卡片/目录结构**。
